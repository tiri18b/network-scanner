# Network Scanner

## Overview
Python-based network scanner for identifying open services on local network hosts.

## Features
- Scans IP ranges
- Detects common IT ports such as RDP, SMB, SQL, HTTP, HTTPS and SSH
- Useful for troubleshooting and basic network discovery

## Technologies
- Python
- TCP/IP
- Socket programming

## How to Run
```bash
python scanner.py
```

## Example
```text
Enter subnet prefix, example 192.168.10.: 192.168.10.
Start host number: 1
End host number: 50
```
